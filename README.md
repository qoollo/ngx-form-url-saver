# FormUrlSaverLib

## Сборка

```bash
ng build form-url-saver-lib
```
Результаты будут помещены в директорию `dist/`

## Публикация

- Выполнить сборку проекта
- Перейти в директорию `dist/`
- Выполнить скрипт публикации

```bash
npm publish
```

## Запуск Unit-тестов

```bash
ng test form-url-saver-lib
```
